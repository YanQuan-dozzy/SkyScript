关于本项目

本工具用于将 SkyStudio 导出的乐谱 .txt 转换为 AutoJS 自动化脚本 .js。

转换核心实现了完整的 SkyStudio 乐谱解析与 AutoJS 脚本生成：

能力说明
编码识别UTF-16 LE BOM 自动识别与转码
谱面解析支持 Json 谱（"songNotes" 数组）和 ABC 谱（<DontCopyThisLine> 头）
BPM 与节拍按 BPM 计算 lNoteTime，自动拆分 1/2/4 倍间隔
琴键映射15 键布局 + 21 键扩展布局，支持多键同按
输出模板3 种可选：按压（单行函数 + 悬浮窗）/ 按压新（gestures 列表 + 35ms 短按 + 进度条）/ 长按（gestures 列表 + 延音 + 音量键控制）
异常拦截非 UTF-16 / BPM 越界 / 空数据 / IO 失败均返回明确错误码

项目说明

  1. 本工具完全本地化处理：源文件经随机重命名后存储，转换过程不依赖第三方服务，文件在 72 小时后自动清理。
  2. 支持 Json 谱与 ABC 谱解析，可选择 3 种输出模板：按压 / 按压新 / 长按。
  3. 单文件支持 .txt 与 .js 任意；批量 ZIP支持 .txt 与 .js 混合压缩；文件夹批量输入服务器侧绝对路径，递归扫描 .txt 与 .js。当源是 .js 时需额外指定"源模板"以便反解析回音符表。
  4. 保存方式可选：下载链接（72h 内有效）/ Web 可访问（可在浏览器直接打开）/ 本地文件夹（资源管理器可见，不会被自动清理）。
  5. 所有临时文件 72 小时后自动清理；管理后台仅限管理员访问，默认账号密码请在部署后立即修改。
